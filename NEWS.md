# rbrsa 0.2.0

* Support for multiple `grup_kod` code and multiple provinces (`il`) in fetch_finturk()
* Support for multiple `grup_kod` code in fetch_bddk()
* A vignette "Getting Started with Turkish Banking Data" that provides an introduction to the package's main features.


# rbrsa 0.1.0

* Initial release with core functionality
* Support for 17 monthly BRSA tables
* Support for 7 quarterly Finturk tables
* Built-in metadata for tables, groups, and cities
* Export functions for RDS, CSV, and Excel formats
